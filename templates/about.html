Created for Udacity's Full Stack Web Developer Project 3

This project creates a working item catalog. Items contain a name, description, and an optional image. Items are added to a category by users who have been authenticated via OAuth with Google, Facebook, or GitHub. Items are stored in a SQLite database and the Item Catalog app supports all CRUD operations.

You can find the source at GitHub.
